# รูปการ์ดนักเตะ (Card Images)

วางไฟล์รูปการ์ดในโฟลเดอร์นี้ (`public/cards/`) — จะเข้าถึงได้ผ่าน URL `/cards/<ชื่อไฟล์>`

## แนะนำการตั้งชื่อไฟล์
ใช้รูปแบบ: `<player-slug>-<tier>.<ext>`

ตัวอย่าง:
- `salah-gold.png`
- `salah-totw.png`
- `haaland-icon.png`

หรือถ้ารูปเป็น "การ์ดเต็มใบ" (มีกรอบ+รูปนักเตะรวมในรูปเดียว) ก็ตั้งชื่อตามนักเตะได้เลย เช่น `salah.png`

## หมายเหตุ
- รองรับ `.png`, `.jpg`, `.webp`
- โครงสร้างนี้ปรับได้ — หลังได้เห็นรูปจริงชุดแรก จะปรับ mapping ใน seed/DB ให้ตรง
- ถ้ามีรูป "กรอบการ์ดตามระดับ" แยกจากรูปนักเตะ แจ้งได้ จะทำระบบ composite (กรอบ + รูป + โลโก้สโมสร) ให้
